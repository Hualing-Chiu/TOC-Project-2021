A Line bot based on a finite state machine

## 創立主旨
因為很常不知道自己三餐該吃什麼而苦惱很久，或是朋友來到成大附近常問我推薦哪些餐廳，就可以直接讓他們加入這個官方帳號選自己想吃的餐廳類型
